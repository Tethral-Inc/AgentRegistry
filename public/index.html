What ACR Does

Skill Catalog

403+ skills indexed from npm, GitHub, and PyPI. Full-text search by name, description, or capability. Quality scores and version tracking.

Security Scanner

20+ threat patterns detect prompt injection, data exfiltration, and code execution. Runs on every skill BEFORE it enters the catalog.

Threat Blocking

Dangerous skills are blocked. Content is redacted. Agents see clear warnings with threat patterns and are advised to uninstall.

Notifications

Agents auto-subscribe to threat alerts for their installed skills. When a skill is flagged, subscribed agents are notified.

Friction Reports

Which external calls cost the most time? Per-target latency breakdown with p95, failure rates, and population baselines.

Network Intelligence

Cross-agent anomaly correlation. When multiple agents from different providers report issues, the signal is amplified.

Every skill is scanned before agents can access it.
